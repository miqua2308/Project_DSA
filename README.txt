# Jungle Escape Game

## Introduction

Welcome to the Jungle Escape Game! In this game, you spawn in a dense jungle with a friendly slime and your mission is to escape. You need to find a key to open a door, be cautious of the Racer monster, and find a boat to win the game.

## Requirements

- Java Development Kit (JDK) 8 or higher

## How to Compile and Run the Game

### Step 1: Download the Source Code

1. Download the source code from the GitHub repository or extract the provided ZIP file to your local machine.
Github: https://github.com/miqua2308/Project_DSA.git

### Step 2: Navigate to the Source Directory

1. Open a terminal or command prompt.
2. Navigate to the directory containing the `Main.java` file. For example:
   ```bash
   cd src/Main/Main.java
   ```

### Step 3: Compile the Source Code

1. Compile the Java source files. Use the following command:
   ```bash
   javac Main.java
   ```
   This command compiles `Main.java` and all the required classes.

### Step 4: Run the Game

1. Run the compiled `Main` class using the following command:
   ```bash
   java Main
   ```

## Game Controls

- **Title Screen:**
  - Use 'W' to move the option up.
  - Use 'S' to move the option down.
  - Press 'ENTER' to select an option.

- **In-Game:**
  - Press 'WASD' to move.
  - Press 'ESC' to open the option menu.
  - Press 'ENTER' to attack.
  - Press 'E' to open status.
  - Press 'P' to pause.

## Game Objective

- Find the key to open the door.
- Avoid or confront the Racer monster.
- Find the boat to win the game.
- You lose if your HP drops to 0.
- You win by reaching the boat and pressing 'ENTER'.

Enjoy the game and good luck escaping the jungle!
